About Sohan Web Development Academy

A focused place for kids and teens to learn real web development in a safe, structured way.

Instructor-led · Project-based

Sohan Web Development Academy is founded and taught by Md. Shafiul Islam Sohan. The program is designed specifically for students aged 10–14, with content carefully matched to their age and learning speed.

Instead of only watching videos, students build real-world projects such as portfolios, quizzes, dashboards and more — using the same tools as professional developers.

  • Friendly explanations with real examples
  • Special focus on problem-solving and logic
  • Regular communication with parents
  • Emphasis on safe, respectful online behavior

Age-Based Learning Plan (10–14 Years)

A clear roadmap so parents know exactly what their child is learning at each age.

Age 10–11 · Foundation
  • Scratch & Code.org logic games
  • Basic computer & typing skills
  • How websites and the internet work
  • Introduction to digital safety
Age 11–12 · Web Start
  • HTML structure (headings, images, links)
  • CSS basics (color, fonts, spacing)
  • First personal and school project websites
  • Simple layout with Flexbox
Age 12–13 · JavaScript Core
  • Variables, functions, loops, conditions
  • DOM & events (click, key, input)
  • Small apps like calculators & quizzes
  • Git & GitHub introduction
Age 13–14 · Modern Web Developer
  • React basics (components, props, state)
  • Intro to Next.js & simple APIs
  • Node.js & MongoDB (beginner level)
  • Publishing and portfolio building
